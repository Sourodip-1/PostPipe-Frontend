# **App Name**: Postpipe Next Migration

## Core Features:

- Next.js Display: Display a welcome message indicating the transition to Next.js.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke a sense of calm and progress.
- Background color: Very light gray (#F0F8FF) for a clean and modern look.
- Accent color: Gentle orange (#FFA07A), analogous to sky blue, used for interactive elements and highlights, providing a warm contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used throughout the page for a modern, neutral feel.
- Simple, centered layout for the welcome message to ensure readability and focus.
